# 🛠️ Welcome to My DevOps Sandbox

This repository is where I bridge the gap between development and operations. It contains a curated collection of my hands-on engineering projects focused on building highly available, scalable infrastructure, writing robust deployment pipelines, and experimenting with cutting-edge cloud-native technologies. 

Whether it's provisioning infrastructure with code, orchestration at scale, or securing a delivery pipeline, this is my playground for solving modern infrastructure challenges. Feel free to explore the code, test the configurations, or adapt these blueprints for your own environments!

---

## 🧰 The Tech Stack & Ecosystem

Here are the primary tools, platforms, and methodologies showcased across these projects:

*   **Cloud Providers:** AWS (IAM, VPC, EKS, EC2, S3) / Azure / GCP
*   **Infrastructure as Code (IaC):** Terraform, OpenTofu, Ansible
*   **Containers & Orchestration:** Docker, Kubernetes (K8s), Helm
*   **CI/CD Automation:** GitHub Actions, GitLab CI, Jenkins
*   **Monitoring & Observability:** Prometheus, Grafana, ELK Stack
*   **Security & Secrets:** HashiCorp Vault, SonarQube, Trivy

## 📂 Project Highlights

Inside this repo, you'll find end-to-end implementations of:
*   **GitOps & Continuous Delivery:** Automated pipelines that build, test, scan, and deploy microservices on every push.
*   **Immutable Infrastructure:** Fully modularized Terraform scripts to spin up production-ready cloud environments from scratch.
*   **Microservices Orchestration:** Kubernetes manifests and Helm charts configured for auto-scaling, self-healing, and service discovery.
*   **Log Aggregation & Metrics:** Centralized observability dashboards tracking cluster health and application performance in real-time.
